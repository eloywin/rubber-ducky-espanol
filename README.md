Es muy intuitivo.
En la carpeta functions se pueden agregar las funciones deseadas.
En code.py puedes llamar a las funciones que seran ejecutadas.
Ya viene con funciones para abrir shell de linux y con funcion para utilizar el teclado.
Tambien viene con un archivo para controlar el led integrado en la board, y una funcion triple blink para avisar que ha comenzado o terminado.
